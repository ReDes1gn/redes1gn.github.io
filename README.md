# redes1gn.github.io

The public landing page for **RēDesign** — drop in a UI screenshot, fan it out
to many frontier AI models at once, and browse a wall of self-contained HTML
reimaginings side by side with the original.

- **Live site:** https://redes1gn.github.io/
- Single self-contained `index.html` (inline CSS/SVG; the only external request
  is the Google Fonts stylesheet). Served straight from `main` via GitHub Pages.

The RēDesign application itself lives in a separate, private repository; this
repo only hosts the marketing page.
